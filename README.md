# African Data Analysis Project

## Overview
This project analyzes a real African dataset from the World Bank using Python in Google Colab.

## Dataset Source
World Bank Africa Indicators

## Objectives
- Load and explore dataset
- Clean data
- Perform data visualization
- Understand trends over time

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations
- Histogram
- Scatter Plot
- Bar Chart
- Box Plot
- Correlation Heatmap
